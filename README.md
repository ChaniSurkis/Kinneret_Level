# Kinneret Level  

### Water Data Visualization Project

Kinneret Level is a data-driven project that retrieves and presents public water-level information from official government data sources, such as data.gov.

The project focuses on displaying up-to-date and historical water data in a clear and accessible way, combining a modern frontend with a structured backend.

## Overview
The purpose of this project is to provide a simple and transparent view of water-related data, specifically water level measurements, based on open government datasets.

The system fetches raw data from public APIs and processes it for presentation to end users.

---

## Data Source
- Public open data from official government platforms (e.g. data.gov)
- Water level measurements and related environmental data
- Read-only consumption of public datasets

---

## Main Features
- Fetching water data from a public government API
- Backend processing and data handling
- Frontend display of water level information
- Separation between client-side and server-side logic
- Extensible design for future analytics and visualizations

---

## Technologies Used
- Frontend: Angular
- Backend: C# (.NET)
- Public Data APIs (Open Government Data)
- REST-based communication

---

## Development Status
The project is in an early development stage.

Current focus areas include:
- Stabilizing data fetching from the public API
- Improving data parsing and validation
- Enhancing data presentation on the client side
